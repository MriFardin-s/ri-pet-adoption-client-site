# Pet Adoption Platform

The Pet Adoption Platform is a dedicated web application designed to connect homeless pets with loving families. Our mission is to simplify the adoption process and provide a safe, user-friendly environment for both pet seekers and pet owners to find their perfect companions.

---

## 🚀 Project Purpose

The primary goal of this platform is to bridge the gap between animal shelters/owners and people looking to adopt. By providing a streamlined digital workflow for adoption requests and pet management, the project ensures a transparent, secure, and delightful experience for animal lovers.

---

## ✨ Core Features

* **Pet Listings:** Browse and discover pets through an intuitive interface with dedicated categories like Dogs, Cats, Birds, and more.
* **Adoption Requests:** Users can easily submit detailed adoption requests for their favorite pets directly through the platform.
* **Secure Authentication:** A robust user login and registration system to keep personal data, listings, and requests safe.
* **Pet Management:** Owners and administrators can seamlessly list new pets, edit details, and manage active profiles.
* **Interactive UI:** Smooth, engaging animations powered by Framer Motion to elevate the overall premium feel of the user experience.
* **Responsive Design:** Fully optimized layouts tailored for a seamless experience across mobile, tablet, and desktop devices.

---

## 🛠️ Main Technologies & Dependencies

### Core Stack
* **Framework:** Next.js
* **Styling:** Tailwind CSS

### Key Dependencies
* **Animations:** `framer-motion`
* **Icons:** `lucide-react`
* **Notifications:** `react-hot-toast`

## 💻 How to Run the Project Locally

Follow these step-by-step instructions to get a copy of the project up and running on your local machine:

### Prerequisites
Make sure you have **Node.js** (v18 or higher) and **npm/yarn** installed.

### Step 1: Clone the Repository
```bash
git clone https://github.com/MriFardin-s/ri-pet-adoption-client-site.git
cd ri-pet-adoption-client-site
```
### Step 2: Install Dependencies
```bash
npm run dev
# or
yarn dev
```
