## Purpose
The Pet Adoption Platform is a dedicated web application designed to connect homeless pets with loving families. Our mission is to simplify the adoption process and provide a safe, user-friendly environment for both pet seekers and pet owners to find their perfect companions.

## Live URL
[Click here to visit the Pet Adoption Platform](https://ri-pet-adoption-client-site.vercel.app/)

## Features
* **Pet Listings:** Browse pets through an intuitive interface with categories like Dogs, Cats, Birds, and more.
* **Adoption Requests:** Users can easily submit adoption requests for their favorite pets.
* **Secure Authentication:** Robust user login and registration system to keep data safe.
* **Pet Management:** Owners can seamlessly list, edit, and manage their pets' profiles.
* **Interactive UI:** Smooth, engaging animations powered by Framer Motion for a premium user experience.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices.

## NPM Packages Used
The following packages were used to build this project:
* `next` - The core framework for the application.
* `framer-motion` - For high-quality animations and transitions.
* `lucide-react` - For modern, clean icon sets.
* `react-hot-toast` - For elegant user notifications and feedback.
* `tailwind` - For efficient and clean CSS class management.
